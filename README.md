## 🛠️ Getting Started

### Prerequisites

Install Node.js before starting:

https://nodejs.org/

### Installation

1. Clone repository

```bash
git clone https://github.com/khanirfan18/finBoard.git
```

2. Enter project folder

```bash
cd finBoard
```

3. Install packages

```bash
npm install
```

4. Start development server

```bash
npm run dev
```
